# NIT Rourkela Roadmap Website

This is a simple website that shows a roadmap to prepare for JEE Main and the target ranks for NIT Rourkela branches.

---

## Features

- Hero section with title and description  
- Roadmap section with 3 phases of preparation  
- Rank section showing target ranks for branches  
- Dark theme with clean design  

---

## How to Use

1. Download or clone the project.  
2. Open the `index.html` file in your web browser.  
3. You can edit the text or styles in `index.html` and `style.css` if you want.
